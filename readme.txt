This Plugin enables you to add a RSS Badge to your sidebar.

== Description ==

This Widget shows big RSS Bagde in your Sidebar. You can choose one the provided Bagdes or
a you can use a random one. (So the user get every click another RSS Badge.)

If you want to have more Badges you can easily copy any PNG, GIF or JPEG images in the rss-icons/
folder of the Badge Feed Link Widget directory. (You will find some other PNG images there.)

* **PHP5!**

== Installation ==

Installation as usual.

1. Unzip and Upload all files to a sub directory in "/wp-content/plugins/".
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Go to your widget admin page and choose a place for your new Widget.
1. You like what you see?
